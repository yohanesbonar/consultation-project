A progressive Node.js framework for building efficient and scalable server-side applications.


### Installation

Need [Node.js](https://nodejs.org/) v14+

```sh
$ npm install -g @nestjs/cli
$ npm install --save
```

how to run the project :

Development
```sh
$ npm run start
```

Watch Mode
```sh
$ npm run start:dev
```

Production
```sh
$ npm run start:prod
```

### Test

```sh
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
